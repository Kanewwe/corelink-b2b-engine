# Linkora B2B 業務流程手冊 (Business Process)

本文件詳述 Linkora v3.2 的業務流程，涵蓋從探勘、聯繫到對帳的完整生命週期。

---

## 1. 核心業務流程 (The Lead Lifecycle)

1.  **探勘階段 (Discovery)**: Member 啟動「製造商」或「黃頁」模式。系統實作 **Four-stage Discovery**。v3.2 新增 **AI Company Intelligence** ✨ 功能，點擊即可生成公司情報摘要與開發切入點。
2.  **分類與標籤 (Classification)**: AI (GPT-4o-mini) 自動將 Leads 依產業進行分類。
3.  **開發與投遞 (Outreach)**: 
    -   **主旨優化**：AI 自動生成 3 個不同策略的主旨（理性/好奇/極簡）。 ✨
    -   **A/B 測試**：同步生成數據型與情感型雙版本內容。 ✨
4.  **成效追蹤 (Analytics)**: 提供漏斗分析與 AI 建議的最佳發信時間。

---

## 2. 協作與交付規約 (Operations)

-   **開發規約**: 全面執行 **VCP (Verify-Commit-Push)**，所有變更必須先在 UAT 驗證後才可合併至 PRD。
-   **角色職責**: Admin 管理系統 API Hub，Vendor 管理旗下 Member 數據。

---

## 3. 預期效益 (Value Analysis)

-   **精準開發**: AI 情報讓業務人員在聯繫前即掌握對方業務痛點。
-   **轉化優化**: 主旨優化與 A/B 測試預期提升開信率 15-20%。
