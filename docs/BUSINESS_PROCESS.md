## 二、詳細業務流程

### 2.1 會員開發流程 (Member Journey)

```
Day 1: 帳號開通
├─ Admin/Vendor 建立會員帳號
├─ 設定方案配額 (leads/emails/templates)
└─ 會員收到開通信件

Day 2-7: 探勘階段
├─ 會員登入 LeadEngine
├─ 選擇探勘模式 (黃頁/製造商)
├─ 輸入關鍵字與市場
├─ 執行背景爬蟲任務
├─ 查看 History 任務進度
└─ 探勘結果自動入庫 (Leads)

Day 8-14: 郵件階段
├─ 進入 Templates 建立開發信
├─ 使用 AI 生成或手動編輯
├─ 進入 Campaigns 執行投遞
├─ 選擇目標 Leads 群組
├─ 設定發送參數 (時間/批次)
└─ 自動化投遞執行

Day 15-30: 追蹤階段
├─ 進入 Analytics 查看成效
├─ 監控開信率/點擊率
├─ 標記有回應的潛在客戶
├─ 手動跟進高意向客戶
└─ 成交後標記轉化狀態
```

### 2.2 廠商管理流程 (Vendor Journey)

```
Month 1: 廠商開通
├─ Admin 與 Vendor 簽訂合作合約
├─ 設定維度計價基礎 (發送/觸及/成交)
├─ Vendor 登入查看控制台
└─ 開始進行開發作業

Month 2-12: 營運階段
├─ 每日查看用量監控儀表
├─ 每週檢視旗下會員活躍度
├─ 每月確認對帳報表
├─ 向 Admin 確認結算金額
└─ 協助旗下會員解決問題

Monthly: 結算流程
├─ 系統自動統計各維度用量
│   ├─ 單次發送 (Outreach): 寄出信件總數
│   ├─ 有效觸及 (Reach): 開信或點擊總數
│   └─ 訂單確定 (Conversion): 標記為成交之 Lead 數
├─ 依照合約單價計算結算金額
├─ Vendor 確認對帳報表
├─ Admin 審核並執行後續流程
└─ 更新 Vendor 帳戶記錄
```

### 2.3 管理營運流程 (Admin Journey)

```
Daily: 系統監控
├─ 查看全系統用量儀表
├─ 監控爬蟲任務佇列
├─ 檢視郵件發送狀態
└─ 處理系統異常警報

Weekly: 業務檢視
├─ 分析各 Vendor 績效
├─ 檢視會員留存率
├─ 調整方案定價策略
└─ 規劃系統優先級

Monthly: 結算管理
├─ 產生各 Vendor 對帳報表
├─ 審核並確認結算金額
├─ 執行付款流程
├─ 更新財務記錄
└─ 分析營收與成本

Quarterly: 策略規劃
├─ 評估市場需求變化
├─ 規劃新功能開發
├─ 調整方案配額與定價
└─ 制定成長目標
```

## 三、關鍵業務指標 (KPI)

### 3.1 營運指標

| 指標 | 計算方式 | 目標值 | 監控頻率 |
|------|---------|--------|---------|
| 月活躍會員數 (MAU) | 當月有登入的會員數 | > 70% | 每月 |
| 會員留存率 | (本月活躍 / 上月活躍) | > 80% | 每月 |
| 平均 leads/會員 | 總 leads / 會員數 | > 50 | 每月 |
| 郵件開信率 | 開信數 / 發送數 | > 25% | 每週 |
| 郵件點擊率 | 點擊數 / 發送數 | > 5% | 每週 |

### 3.2 財務指標

| 指標 | 計算方式 | 目標值 | 監控頻率 |
|------|---------|--------|---------|
| 月經常性營收 (MRR) | 所有會員月費總和 | 成長 10%/月 | 每月 |
| 客戶獲取成本 (CAC) | 行銷費用 / 新會員數 | < NT$1,500 | 每季 |
| 客戶生命週期價值 (LTV) | 平均月費 × 留存月數 | > NT$9,000 | 每季 |
| LTV/CAC 比率 | LTV / CAC | > 3:1 | 每季 |
| 毛利率 | (營收 - 成本) / 營收 | > 60% | 每月 |

### 3.3 技術指標

| 指標 | 目標值 | 監控頻率 |
|------|--------|---------|
| 系統可用性 (Uptime) | > 99.9% | 即時 |
| API 平均響應時間 | < 200ms | 即時 |
| 爬蟲成功率 | > 95% | 每日 |
| 郵件送達率 | > 98% | 每日 |
| 錯誤率 | < 0.1% | 每日 |

## 八、結論

Linkora B2B 引擎透過三層角色架構 (Admin/Vendor/Member)，建立可規模化的 B2B 探勘與開發平台：

1. **Admin** 專注平台營運與變現
2. **Vendor** 透過批發模式創造利潤
3. **Member** 以低成本獲取高品質潛在客戶

預期在 6 個月內達到：
- 200+ 付費會員
- $10,000+ 月經常性營收
- 85%+ 會員留存率
